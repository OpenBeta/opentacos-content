# Description
This is a mixed 3 pitch route. The first pitch is fully bolted, the second pitch is mostly trad with a few bolts, the third pitch is mostly bolted. This line allows multiple connections with other routes higher up.

# Location
On the SE face wall this climb starts on top of a small ramp.

# Protection
3 pitches of sport/trad. P1 fully bolted.
