# Description
Climb begins next to a large boulder...possible 'direct' start up and over.  Using tree as belay anchor, boulder up through the notch between large boulder and tower.  Clip a single bolt to traverse across to the hands+ crack.  Protects with a set of Camalots incl. #4, #5 .  1/2" stainless glue-ins on top.  Not much room up there.  Front side remains unclimbed.

# Location
Hike to the top/back side of Trespass Tower, the tower closest to the lower parking.  Belay from tree.

# Protection
Set of cams.
