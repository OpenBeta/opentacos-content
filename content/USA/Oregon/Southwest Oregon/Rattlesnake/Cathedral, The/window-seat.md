# Description
The bolted crack straight above the Window. Starts on the right of the Window. The crux is a few painful jams/locks on the steepest part of the route. Traverse left to the anchor.

# Location
Inside the Cathedral.

# Protection
STICK CLIP. Four bolts. Shares an anchor with The Groove. Keep the last bolt clipped if TRing.
