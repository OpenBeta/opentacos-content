# Description
This is an open book climb that includes an easy roof. Holds can be found among the moss and dirt at the base of the route.

# Location
The climb begins just to the right of a madrone tree at the very end of the secret cleft on the left side. A top rope can be set up by walking to the top where there is a bolted anchor.

# Protection
Bolted anchor
