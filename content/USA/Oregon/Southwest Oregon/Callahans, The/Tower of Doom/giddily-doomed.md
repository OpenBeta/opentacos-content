# Description
Up hill from Bite the Edge in the middle of the SW buttress locate the left-slanting off-width crack. Start a few feet left (looking up) of the off-width following bolts to the top.

# Location
Refer to Rock Climbing Western Oregon Vol. 2: Umpqua map page 66-65, map 7 128, and GPS coordinates in UTM's page 127.

# Protection
Fixe SS 3/8 double expansion, no chains
