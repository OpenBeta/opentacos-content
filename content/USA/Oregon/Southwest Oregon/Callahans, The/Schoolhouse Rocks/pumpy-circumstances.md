# Description
Refer to Umpqua Guide topo S, pp. 122.

From first bolt move right on narrow ledge with more difficult climbing into a cave above. Climb crux move from cave (5.10c) following huecos and pockets. Finish second crux on thin slab friction to summit. Rappel from chains or scramble off the backside.

# Location
Refer to Umpqua guide map 5, pp. 117.

Route climbs inside face of east schoolhouse.

# Protection
bolts, rappel chains
