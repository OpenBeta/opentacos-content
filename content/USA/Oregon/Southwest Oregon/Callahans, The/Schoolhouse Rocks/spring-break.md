# Description
Refer to Umpqua guide topo R, pp.121.

After first clip move right on friction slab to head wall with small ledges. After headwall finish on easier slab merging with Summer Vacation.

Rappel from chains, or scramble down backside.

# Location
Refer to Umpqua guide map 5, pp. 117.

This is the middle bolt line up the left Schoolhouse Rock.

# Protection
bolts, rappel chains
