# Description
Follow the bolt line up some thin face moves into finger cracks and some tricky stemming left of the broken column.  Pull the roof using crimps and negotiate better feet after the overhang.  Climb the remaining technical slab across the diagonal crack to the anchors shared with “Left Crimp”, 2 bolts.

# Location
Second bolted route from the far left column. Listed as 2.) in the route description photo.

# Protection
Reachy first bolt but well protected through the crux moves. Bolted to anchors.
