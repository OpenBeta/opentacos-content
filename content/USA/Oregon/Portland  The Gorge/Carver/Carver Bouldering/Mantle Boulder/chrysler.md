# Description
Sit-start matched on the “Chrysler” crimp and make a big pull to a large right-facing ledge. Pull past the bulge using a combination of crimps and slopers then waltz up the slab.

# Location
This is at the north point of the boulder. Start with another boulder ~4‘ behind you.

# Protection
1-2 pads
