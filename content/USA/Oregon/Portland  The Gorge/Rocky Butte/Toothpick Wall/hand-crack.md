# Description
A slabby climb featuring a prominent crack. There is a tree right up top to use as an anchor

# Location
If you walk past the tree anchor for BlueBerry Jam to the ledge, and continue along the cliffside, the top is at the next large tree right along the cliffside

# Protection
Pro to 1", Tree anchor right above the crack
