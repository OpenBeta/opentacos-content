# Description
Climb up the shallow corner on dirty small holds to the crux bulge, pull a reachy move up and climb into the dihedral. lots of water rushes down this area during the winter so be prepared for a dirty, messy climb through the last section. bring a brush

# Location
Probably the most obvious anchor point on toothpick wall. NE end of wall. set up on the tree. no fixed anchor. can be used to climb Vertical Therapy if you add a directional to keep rope out of the crack

# Protection
TR - pro is thin at best
